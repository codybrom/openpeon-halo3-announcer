# Announcer (Halo 3) — OpenPeon Sound Pack

<p align="center"><em>"Capture the flag."</em></p>

A [peon-ping](https://github.com/PeonPing/peon-ping) sound pack featuring the iconic **Halo 3 Multiplayer Announcer** voice and some sound effects.

## Install

```bash
peon packs add codybrom/openpeon-halo3-announcer
peon packs use halo3_announcer
```
